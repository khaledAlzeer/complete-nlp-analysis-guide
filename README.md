# NLP Linguistic Semantic Studio 🧠

> A complete NLP documentation website covering Linguistic Analysis Levels, Semantic Analysis, NLTK, spaCy, Parsing, NER, WSD, and Sentiment Analysis with practical Python examples.

---

## 📖 Overview

NLP Linguistic Semantic Studio is an interactive educational website designed as a comprehensive reference guide for Natural Language Processing (NLP).

The project combines theoretical concepts with practical Python examples using both NLTK and spaCy, covering the complete journey from lexical processing to semantic interpretation and sentiment analysis.

---

## 🎯 Topics Covered

### Chapter 1 — Linguistic Analysis Levels

#### Level 1 — Lexical Analysis

* Tokenization
* Sentence Tokenization
* Word Tokenization
* Stop Words Removal

#### Level 2 — Morphological Analysis

* Porter Stemmer
* Lancaster Stemmer
* Snowball Stemmer
* Lemmatization
* Stemming vs Lemmatization

#### Level 3 — Syntactic Analysis

* POS Tagging
* Chunking
* Shallow Parsing
* Constituency Parsing
* Dependency Parsing

#### Level 4 — Semantic Analysis

* Named Entity Recognition (NER)
* Semantic Class Disambiguation (SCD)

#### Level 5 — Pragmatic Analysis

* Word Sense Disambiguation (WSD)
* Lesk Algorithm
